Readme for Svartberget_forcing_2019.csv

Kersti Haahti, Luke 2020-05-14

yyyy, mo, dd, hh, mm: datetime [UTC + 1.0]
doy: Day of year [days]
Prec: Precipitation [mm/30min]
P: Ambient pressure [Pa]
Tair: Air temperature [degC]
Tdaily: Daily air temperature [degC]
U: Wind speed [m/s]
Ustar: Friction velocity [m/s]
H2O: Ambient H2O [mmol/mol]
CO2: Ambient CO2 [ppm]
Zen: Zenith angle [rad], (lat = 64.26, lon = 19.77)
LWin: Downwelling long wave radiation [W/m2]
diffPar: Diffuse PAR [W/m2] 
dirPar: Direct PAR [W/m2]
diffNir: Diffuse NIR [W/m2] 
dirNir: Direct NIR [W/m2]
Tsoil: Soil surface layer temperature [degC]]
Wliq: Soil surface layer moisture content [m3 m-3]
X: phenomodel delayed temperature [degC]
DDsum: degreedays [days]
