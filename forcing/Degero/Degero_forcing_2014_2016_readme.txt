Readme for Degero_forcing_2014_2016.csv

Kersti Haahti, Luke 2020-01-20
Samuli Launiainen, Luke 2020-04-14

yyyy, month, day, hour, minute: datetime [UTC + 1.0]
doy: Day of year [days]
Prec: Precipitation [kg m-2 30min-1 == mm 30min-1] - divide by 1800.0 to get [kg m-2 s-1 = mm s-1]
P: Ambient pressure [Pa]
Tair: Air temperature [degC]
Tdaily: Daily air temperature [degC]
U: Wind speed [m/s]
Ustar: Friction velocity [m/s]
H2O: Ambient H2O [ppth] - divide by 1000 to get mol/mol
CO2: Ambient CO2 [ppm] - set constant!
Zen: Zenith angle [rad], (lat = 64.18, lon = 19.55)
Rg: Global radiation [W/m2]
LWin: Downwelling long wave radiation [W/m2]
LWout: Upwelling long wave radiation [W/m2]
diffPar: Diffuse PAR [W/m2] 
dirPar: Direct PAR [W/m2]
diffNir: Diffuse NIR [W/m2] 
dirNir: Direct NIR [W/m2]
Rnet: Net radiation [W/m2]
X: phenomodel delayed temperature [degC]
DDsum: degreedays [days]
